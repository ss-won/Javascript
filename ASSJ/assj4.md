# 알쓸신자 Part

## Truthy Falsy?
<br/>

### 1.  `Truthy` and `Falsy`
- Truthy = true에 가까운 값
- Falsy = false에 가까운 값
- `Null`, `Undefined`, `0`, `''(공백)`, `NaN(Not a Number)`은 모두 언제나 falsy한 값이다.

<hr>

### 2.  Null Checking에서의 활용
- Null Checking이란, 어떤 값이 Null 또는 Undefiend인지 식별해내는 것을 말한다.
- Null, Undefined는 언제나 Falsy한 값이므로 부정 연산자(`!`)를 가지고 Checking 할 수 있다.
